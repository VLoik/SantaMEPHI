# SantaMEPHI
Бот для соц.сети vk.com для проведения игры "Тайный Санта"  

Игра проводится в канун нового года. Пользователи регитрируются в базе путем личного сообщения боту, указывают свою учебную группу и другую информацию о себе. После старта игры пользователи анонимно распределяются по парам, после чего "дарителю" отправляется ссылка на профиль "одариваемого". Каждый пользователь анонимно получает подарок от "дарителя" и анонимно дарит подарок "одариваемому".  
Также предоставляется возможность анонимного обмена сообщениями в парах.
